# dqnid
## My linux rice
System designed to be simple and usable while remaining full of features.
* Distro : `Arch`
* Fonts : `Source Code Pro, Waffle, Terminus`
* WM : `I3`
* Hotkey daemon : `sxhkd`
* Bar : `Polybar`
* Widgets : `Rofi`
* MPD client : `ncmpcpp`
* GUI FM : `Nemo`
* Terminal FM : `Ranger`
* Compositor : `Picom`
